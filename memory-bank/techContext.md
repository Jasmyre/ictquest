# Tech Context

## Dependencies
- Testing: 
  - `jest`
  - `@types/jest`
  - `@testing-library/react`
  - `@testing-library/jest-dom`
  - `@testing-library/user-event`
  - `msw`

## Development Setup
- Testing: Run tests with `npm test`
- Test coverage reports: `npm test -- --coverage`

## Technical Constraints
- Test coverage must be at least 80%
- All components must have snapshot tests
- API routes must have integration tests
